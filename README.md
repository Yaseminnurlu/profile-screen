# Profile Header with Overlay Card

This project is a **profile screen layout** using **Jetpack Compose Box layering**, following Material 3 design principles. The layout features a classic **profile card overlap effect**.


## Features

* **Header background**: light pink
* **Circular avatar**: clipped with `CircleShape`, centered in header
* **Overlay card**: partially overlapping the header
* **Follow button**: changes text when clicked
* **Material 3 components**: `Card`, `Button`, `Surface`, `Text`, `Icon`
* **Modifier techniques demonstrated**:

  * `clip(CircleShape)`
  * `offset()`
  * `zIndex`
  * `shadow / elevation` via Card
  * `fixed size` (`.size(100.dp)` for avatar)
